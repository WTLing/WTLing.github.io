Linux命令

基本概念及操作

【Tab】 补全代码

touch name 创建一个名为name的文件

pwd 查看当前所在目录

find 查找某文件

【ctrl+c】 终止程序

【ctrl+a】 将光标移到行头

【ctrl+e】 将光标移到行末

【ctrl+k】 从光标处删除到行末

man 查看命令的手册

通配符的使用

touch love_{1..10}_shiyanlou.txt 快速生成1-10的对应的十个文件

ls *.txt 查找所有txt的文件

banner 输出图形字符

  • sudo apt-get install sysvbanner*

用户及文件权限管理

who命令查看当前用户信息

su 切换到用户user

sudo adduser 新建一个为user的用户**

groups 查看user所属的用户组

Sudo usermod -G sudo lilei 使用户获得sudo

ls -l 列出所有文件及其详细

ls -l 列出指定文件详细

sudo chown 将文件所有者变更

chmod 777 改变指定文件的权限

目录结构及对文件的基本操作

cd - 返回上一次的目录

cd ~ 返回当前用户的home

mkdir mydir 创建目录

mkdir -p father/son/grandson 创建多级目录

cp 文件 路径 将文件复制到指定路径下

cp -r 文件 路径 递归复制

rm 文件名 删除文件

rm -r family 删除目录

rm -f test 强制删除，针对权限不足的情况

mv 文件名 目的目录 移动文件到指定目录下

mv 旧文件名 新文件名 重命名

cat 文件名 打印文件内容到标准输出

cat -n passwd 显示行号

more/less 文件名 分页查看文件

file 文件名 查看文件的类型

VIM

dw 可以删除从光标处删除一个词至词的末尾

文件打包和解压缩

zip -r -9 -q -o shiyanlou.zip /home/shiyanlou/Desktop

-r 表示递归打包包含子目录的全部内容

-q 安静模式，不向屏幕输出信息

-o 表示输出文件

unzip shiyanlou.zip 将文件解压到当前文件夹下

Unzip shiyanlou.zip -d ziptest 将文件解压到指定文件夹下

  • zip：
  • 打包 ：zip something.zip something （目录请加 -r 参数）
  • 解包：unzip something.zip
  • 指定路径：-d 参数
  • tar：
  • 打包：tar -cf something.tar something
  • 解包：tar -xf something.tar
  • 指定路径：-C 参数
