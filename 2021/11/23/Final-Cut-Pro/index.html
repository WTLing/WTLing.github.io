Final Cut Pro

关系

事件是所有素材存放的地方

项目可以理解为时间线

一般 1080p 25i or 4K 25i

编码器使用主流Apple ProRes 422 ，大项目可以使用Apple ProRes 4444

素材目录分类

一定要养成事件归类的习惯

媒体整理三元素：关键词、分角色、片段过滤

关键词

要确保每个素材都写好关键词

视频粗剪快捷键

Command + / - 素材缩放

I 入点 O出点

F 个人收藏 U 取消收藏

L 播放 K暂停 J倒放 J 和 L 按几次就是几倍速

K + J 一帧一帧倒放

X 选择整条素材

；上一个编辑点 ‘ 下一个编辑点

分角色

分角色就是给同类素材分配一个颜色即一个轨道

三点剪辑法

入点、出点、编辑点

W 插入 根据播放头的位置来插入（会增加时长）

E 追加 出现在时间线尾部（会增加时长）

D 覆盖

Q 连接 把两条素材连接在一起 想单独拖动按住 `

M 打标记点

A 选择

T 修建 Command + T 建立蒙太奇

Control + T 建立标题

P 可以减少素材长度 并流出灰色空隙

R 范围选择

Command R 变速

Fn D 删除留灰

按住 B不放 切割

Command B 切割

Command + Shift + B 全部素材切割

Option + 【 减去播放头左边的素材

Option + 】 减去播放头右边的素材

shift + ？ 试听前后两秒

横版照片 可以在效果器里空间符合中选择填充 再 修改Y轴即可

竖版照片 可以option 按住素材往上 复制一层素材 对底层的素材放大，效果其中 高斯模糊

匹配颜色 调节白平衡
